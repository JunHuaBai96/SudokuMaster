# SudokuMaster - 多功能数独游戏

SudokuMaster是一个功能丰富的网页版数独游戏，支持多种数独类型和游戏模式，为玩家提供丰富的数独解谜体验。

## 功能特点

1. 多种数独类型：
   - 标准数独（9x9网格）
   - 迷你数独（4x4和6x6网格）
   - 超大数独（12x12和16x16网格）
   - 不规则数独
   - 对角线数独

2. 游戏功能：
   - 动态生成数独谜题
   - 提示功能（每局游戏3次机会）
   - 错误检查（每局游戏3次机会）
   - 计时功能
   - 最佳时间记录
   - 高亮显示相关单元格
   - 智能数字键盘

3. 用户界面：
   - 清晰直观的界面设计
   - 响应式布局，适配各种设备
   - 简洁的操作方式

## 使用方法

1. 打开`index.html`文件即可开始游戏
2. 选择想要挑战的数独类型
3. 点击"新游戏"开始游戏
4. 使用鼠标点击空格并通过数字键盘输入数字
5. 需要帮助时可以使用提示功能
6. 填错数字会减少错误机会
7. 完成游戏后可以查看是否创造新的最佳时间记录

## 技术实现

- 使用HTML5、CSS3和JavaScript ES6+开发
- 采用面向对象的编程方式
- 使用LocalStorage保存最佳时间记录
- 响应式设计适配不同屏幕尺寸
