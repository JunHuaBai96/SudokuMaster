游戏玩法介绍

1. 数独类型介绍

标准数独 (9x9)

经典的9x9数独游戏，将数字1-9填入9x9的网格中，使得每行、每列和每个3x3的小方块中的数字都不重复。

迷你数独 (4x4)

使用1-4的数字，填入4x4的网格中。每行、每列和每个2x2的小方块中的数字都不能重复。适合初学者和快速游戏。

迷你数独 (6x6)

使用1-6的数字，填入6x6的网格中。每行、每列和每个2x3的小方块中的数字都不能重复。难度介于4x4和9x9之间。

超大数独 (12x12)

使用1-12的数字，填入12x12的网格中。每行、每列和每个3x4的小方块中的数字都不能重复。对于有经验的玩家提供更大的挑战。

超大数独 (16x16)

使用1-16的数字（1-9和A-G），填入16x16的网格中。每行、每列和每个4x4的小方块中的数字都不能重复。最具挑战性的数独类型。

不规则数独

基于9x9网格，但小区域形状不规则。每个不规则区域内的数字1-9不能重复，同时保持行列不重复的规则。需要更灵活的思维方式。

对角线数独

在标准9x9数独的基础上，增加了对角线约束。两条主对角线上的数字也不能重复，增加了额外的解题难度。

2. 游戏规则

  • 每次开始新游戏时，你有3次填错的机会和3次提示的机会
  • 填错数字会减少一次机会，用完所有机会游戏结束
  • 使用提示功能会减少一次提示机会
  • 完成游戏后会记录你的最佳时间

3. 游戏操作

  • 选择单元格：点击空白格子进行选择
  • 填写数字：使用屏幕上的数字键盘输入数字
  • 高亮功能：
    • 选中格子时会自动高亮显示相关的行、列和小方块
    • 点击已填写的数字时，会高亮显示所有相同的数字
    • 选择空格时，数字键盘上不可用的数字会自动变灰
  • 获取提示：选中空格后点击"提示"按钮
  • 重置游戏：点击"重置"可以清除所有填写的数字

4. 完成游戏

成功填写所有空格且符合规则时，游戏结束。系统会显示完成时间和是否打破最佳记录。
